# Ambient Audio Files

Drop your audio files here. Supported formats: mp3, ogg, wav, m4a

Naming convention — file name must match the scene ID:
  cosmic.mp3   → Cosmic Void scene
  forest.mp3   → Dark Forest scene
  ocean.mp3    → Deep Ocean scene
  ember.mp3    → Ember & Fire scene
  arctic.mp3   → Arctic Wind scene
  lofi.mp3     → Lo-fi Study scene

The app will automatically detect and use the file.
If a file is missing, the synthesized fallback plays instead.
